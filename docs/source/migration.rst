Con el fin de crear una tabla con unos tipos de datos adecuados y eficientes para el almacenamiento de nuestra base de datos, debemos realizar una inferencia del tipo de dato a partir de los Dtypes del dataframe de Pandas que se nos entrega. De aquello se encarga la función infer_sqlalchemy_type(), que recorre todas las columnas del dataframe determina las condiciones que se cumplen para la asignación de determinados tipos de datos.
